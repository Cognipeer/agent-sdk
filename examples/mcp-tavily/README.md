# MCP Tavily example

Shows using MCP tools via LangChain MCP adapters.

## Run

From the `examples/` directory:

```bash
# Set your API keys
export TAVILY_API_KEY=tvly-...
export OPENAI_API_KEY=sk-...

# Run the example
npm run example:mcp-tavily
```

Or directly:
```bash
TAVILY_API_KEY=... OPENAI_API_KEY=... npx tsx mcp-tavily/mcp_tavily.ts
```
