# Summarize Context Example

Forces heavy tool output and demonstrates context summarization and recovery via get_tool_response.

## Run

From the `examples/` directory:

```bash
# Set your API key
export OPENAI_API_KEY=sk-...

# Run the example
npm run example:summarize-context
```

Or directly:
```bash
OPENAI_API_KEY=... npx tsx summarize-context/summarize-context.ts
```
