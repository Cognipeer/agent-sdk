# Prompts and Planning

The canonical prompt reference now lives in [API Prompts](/api/prompts).

Key points:

- planning is adaptive, not mandatory on every task
- `write` creates or fully rewrites a plan
- `update` is the normal operation after a plan already exists
- `expectedVersion` protects updates against stale writes
